str使用技巧
=

1. 将字符串中的字母与数字提取出来
```python
x = '%1234a#'
x = ''.join(filter(str.isalnum, x))
x
output: '1234a'
```
只保留数字： `str.isdigit`       只保留字母： `str.isalpha`
使用自定义规则需要借助 `lambda`
如保留#，去掉%
```python
x = '%1234a#'
x = ''.join(filter(lambda i: i in ['#'] or i.isalnum(), x))
x
output: '1234a#'
```
